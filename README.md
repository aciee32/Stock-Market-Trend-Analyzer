# Stock Market Trend Analyzer 📈
An interactive web application that uses **Deep Learning (LSTM)** to analyze and predict stock market trends.

## Features
- Real-time data fetching via **Yahoo Finance API**.
- Technical Analysis with **100-day & 200-day Moving Averages**.
- Predictive modeling using a **Stacked LSTM Neural Network**.
- Interactive dashboard built with **Streamlit**.

## How to Run
1. Clone the repo: `git clone https://github.com/your-username/your-repo-name.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

NOTE:
This project is for educational purposes. While it leverages the most recent data available, the stock market is influenced by unpredictable news events that historical data cannot fully account for. Use this tool for trend analysis, not as absolute financial advice!
